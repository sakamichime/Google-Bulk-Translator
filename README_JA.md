# Google画像一括翻訳

[English](README_EN.md) | [中文](README.md) | [日本語](README_JA.md)

Google翻訳の画像機能をベースに実装された、画像を一括翻訳するChromeブラウザ拡張機能です。

## 機能特徴

- **一括アップロード**：複数の画像を一度に選択して翻訳可能
- **自動ダウンロード**：翻訳完了後、翻訳済み画像を自動ダウンロード
- **多言語インターフェース**：中国語、英語、日本語の3言語に対応
- **画像管理**：追加した画像の削除、ドラッグ＆ドロップでの翻訳順序調整に対応
- **接続テスト**：Google翻訳サービスが利用可能かどうかを検出する接続テスト機能を内蔵
- **進捗表示**：翻訳の進捗状況とステータスをリアルタイムで表示

## インストール方法

### ソースからインストール

1. 本リポジトリをダウンロードまたはクローンします
2. Chromeブラウザを開き、拡張機能管理ページにアクセスします
   - アドレスバーに `chrome://extensions/` と入力
   - またはメニューから：その他のツール → 拡張機能
3. 右上の「デベロッパーモード」を有効にします
4. 「パッケージ化されていない拡張機能を読み込む」をクリックします
5. 本プロジェクトのあるフォルダを選択します

## 使用方法

### 基本的な使い方

1. **翻訳ページを開く**
   - ブラウザツールバーの拡張機能アイコンをクリック
   - または手動で [Google翻訳画像ページ](https://translate.google.com/?op=images) にアクセス

2. **画像を追加**
   - 「画像を選択」ボタンをクリックして翻訳したい画像を選択
   - または画像をアップロードエリアに直接ドラッグ＆ドロップ

3. **画像を管理**
   - 画像リストの「×」ボタンをクリックして単一の画像を削除
   - 画像をドラッグして翻訳順序を調整

4. **翻訳を開始**
   - 「翻訳開始」ボタンをクリック
   - 翻訳完了を待ち、翻訳済み画像が自動的にダウンロードされます

### インターフェースの説明

- **言語切り替え**：パネルヘッダーの右側でインターフェース言語を選択（中国語/英語/日本語）
- **翻訳ページを開く**：Google翻訳画像ページを素早く開く
- **接続テスト**：現在のネットワークがGoogle翻訳サービスに正常にアクセスできるか確認
- **ステータス表示**：現在が正しい翻訳ページ上にあるかどうかを表示

## 注意事項

1. 本拡張機能は **Google翻訳画像ページ** で使用してください。テキスト翻訳ページではありません
2. 翻訳済み画像はブラウザのダウンロードディレクトリ内の「Google翻訳」フォルダに保存されます
3. 翻訳に失敗した場合は、ネットワーク接続を確認するか、接続テスト機能をお試しください

## 対応画像フォーマット

- PNG
- JPG/JPEG
- WebP
- GIF

## 技術スタック

- Chrome Extension Manifest V3
- Vanilla JavaScript
- CSS3

## ディレクトリ構造

```
├── manifest.json        # 拡張機能設定ファイル
├── background/          # バックグラウンドサービススクリプト
│   └── background.js
├── content/             # コンテンツスクリプト
│   ├── content.js       # メインロジック
│   └── content.css      # スタイルファイル
├── icons/               # 拡張機能アイコン
│   ├── icon16.png
│   ├── icon48.png
│   └── icon128.png
└── popup/               # ポップアップページ（予備）
    ├── popup.html
    ├── popup.js
    └── popup.css
```
